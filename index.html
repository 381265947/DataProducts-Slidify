Here's the code

Notice that spaces are handled properly because we live in the future.

text <- "1, 2, 3, 4";
nums <- sapply(strsplit(text, "\\s*,\\s*"), as.numeric);
avg <- mean(nums);
avg;

## [1] 2.5

Here's the visualization

plot(nums);
abline(h = avg);
